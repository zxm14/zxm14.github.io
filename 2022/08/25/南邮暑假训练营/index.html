摄影概论与基础：快速上手摄影和了解校媒的摄影艺术

摄影流派有哪些

风光摄影

给人带来美的享受是最为全面的，从发现美到与读者见面的全过程给予感官与心灵的愉悦，能够在一定的主题思想表现中，以相应的内涵使人在审美中平添情绪。
风光摄影就是以镜头语言描绘自己对土地与山河的热爱。

包含

  • 自然风光
  • 城市风光

特点

  • 视觉冲击力强（多用广角镜头即短镜头拍摄，画面在突出中央主体和前景的同时拥有广泛的背景，可以在较小的环境里拍摄较多的视觉景物）
  • 色彩层次丰富（Lightroom软件，学习调色技术）
  • 考验摄影师功底、体力

找图软件

  • 500PX

人像摄影

  • 刻画与表现被摄者的具体相貌和神态
  • “形神兼备”
  • 在创意、构图、画质、美颜及体型等要素中取得平衡
  • 唤起观众的情感（传递出模特本身之外更多的东西）

拍摄特点

  • 需要情感的有效沟通和引导，展现人物天性，与被摄者互动性强
  • 注重抓拍

人文摄影（包括街头摄影与新闻摄影）

  • 人文：人类社会的各种文化现象
  • 人文摄影：以人或人的活动为拍摄对象的摄影
  • 触及读者的心灵深处，使读者在爱与关怀、崇敬的情感下与读者产生共鸣
  • 景深：聚焦完成后，焦点前后的范围内所呈现的清晰图像的距离

拍摄特点

  • 纪实性、有情感、有故事
  • 思想较为深刻
  • 构图、色彩、机遇同样要求高
  • 摄影师是记录者而不是介入者（与人像摄影的区别）

创意摄影

  • 前期创意摄影：前期拍摄时，通过一些拍摄技法，表达出独特的创意想法（错误、大小透视）

  • 后期创意摄影：后期通过photoshop(ps)进行一些创意合成。

部分创意

  • 光绘摄影：用光源作为绘画手段的摄影创作方式，通过相机的长曝光，记录下光源的运动轨迹（快门速度较慢）

  • 利用逆光，构造剪影

    • 剪影：摄影中当光线直射景物时，在其背光面形成暗调景象，类似剪纸效果

  • 运用多样小道具

  • 要素重组

电影艺术与摄影艺术

  • 相似性：平面展开&时间限定空间展开
  • 欣赏电影：提升审美和拍照技术（潜移默化）

为什么要学摄影

我们不止用相机拍照，我们带到摄影中去的是所有我们读过的书、看过的电影、听过的歌、走过的路、爱过的人。——Ansel Adams

艺术是互通的，除学到构图、曝光、色调等摄影技术，还有观察世界的角度，对事物的理解，某种心境和经历等。摄影是感知世间万物的方式，它表达了自己的思想，而这些感知和表达的背后来源于摄影师自身的经历。

对于摄影师来说，重要的不是摄影师的镜头，而是相机背后的那颗头。

摄影基础知识（曝光三要素）

光圈大小（景深-，进光量+）

大光圈可以使对焦点清晰，而画面其他部分相对模糊，适用于人像、夜景、快门摄影

小光圈可以使对焦点清晰，而画面的其他部分也清晰，适用于合照、风光、慢门摄影

快门速度（脚架-）

快门速度越快，则越适合拍摄体育运动场景和动物活动场景

快门速度较慢，则越适合拍摄流水、车流等场景，能展现出水流和车流的动感。

ISO感光度

ISO的多少主要影响画面的纯净度。

ISO越低感光度越低，画面越纯净，但是夜晚容易欠曝

ISO越高感光度越高，产生的电子杂讯（噪点）也越多，画面质量相对较差

  • 明亮度：调整颜色的亮度，调整具体的一种颜色

人物专访&图文短句

人物专访

过程

  1. 主动接稿
  2. 对采访对象进行了解、背景调查联系被采访对象询问一些相关个人资料，确定对方接受采访后，敲定采访时间、地点、形式等相关细则。
  3. 寻找个人简介关键词
  4. 列出采访提纲（根据关键词）
  5. 保证采访问题有话可达

注意点

  • 全程录音

  • 随机应变，不拘泥于提纲

  • 采访时采用聊天交流式，平等关系

  • 引导挖掘

  • 轻言慢语，不退不逼，帮助思考回忆

  • 记录灵感和关键点

写稿

  • 深度挖掘受访者的特点，围绕特点来写，如描写形象或引用原话（与众不同）
  • 搭建框架撰写
  • 阐述特点，有佐证，形象更加真实
  • 语言承转上不能一成不变
  • 遇到有关奖项职称，需向受访者具体确认
  • 文章单薄时可补充采访，或采访朋友、家人、老师等

收尾

  • 修改
  • 人物专访特点
    • 需要和受访者保持沟通状态，以便确认稿件细节。
    • 审核尾声时，需发给受访人。

步骤

  1. 根据时事热点或灵感，或者借鉴其他优秀公众号，提出策划案
  2. 确定立意，进一步具化想法，构思推送确定创作周期（征集类稿件提前写好征集稿）
  3. 与视觉中心，办公室等其他部门合作，根据时间完成稿件

活动特报

特点

  • 最后传达主要思想
  • 清晰叙述事件
  • 内容真实准确
  • 新鲜有价值
  • 时效性强
  • 简明扼要

主要步骤

1.列采访提纲

  • 明确对象
  • 采访什么
  • 问题适量
  • 类型全面

2.现场采访

  • 选择不同的采访对象
  • 提前找到受访者
  • 注意礼貌
  • 表明身份和目的
  • 录音
  • 拍采访照

3.整理成文

  • 提炼精简原话

前期准备

  1. 获取相应材料，对活动进行基础了解、背景调查，明确活动内容、主题、流程、出席人员等
  2. 根据不同的活动类别列采访提纲（明确重点，问题全面，衔接自然，条理清晰，由浅入深，逻辑性）
  3. 确定对接人，敲定活动时间、地点，提前确定被采人员，联系视觉中心到现场进行拍摄
  4. 现场采访未必能全部记录，提前获取发言稿
  5. 提前写好预稿，开头结尾、文章框架提前拟，减轻当天负担

采访进行

  1. 带好工作证，提前抵达活动地点，确定并对接好被采人员所在位置，以便活动结束后进行采访
  2. 活动过程中做好活动记录，根据现场情况灵活增删被采问题，随机应变
  3. 活动结束，与被采人员进行采访，做好重点记录、采访录音，可先交流确定内容，过程中面带微笑，不卑不亢，亲和力，宽泛话题引入，深挖细节，实质性话题，替换词汇，记得感谢
  4. 留下联系方式，尤其是老师领导，初稿写完后询问修改意见，缺失确认补采

整理成文

  • 趁热打铁，整理挑选采访录音和现场图片
  • 理清写作大纲思路
  • 根据现场情况及采访内容成稿，还原，不可过分加工，但要采用书面化用语
  • 核对被采对象的姓名、职务等相关个人信息（**组织，学院，专业，班级等在新闻稿里一定要写==全称==**）

注意

  • 活动要体现主题，参与者的先后顺序要理清
  • 活动程序分清主次
    • 上下半场
    • 不同地点
  • 注意细节
    • 特殊之处
    • 切入点
  • 报道角度多样
    • 选择不同的采访对象，倾听不同想法

写作

  • 结构
    • 按事实重要性递减
    • 按时间先后的顺序
    • 按逻辑关系安排
    • 按时间和逻辑相结合安排结构
  • 写作要求
    • 总结概括活动主要内容
    • 解释和深化活动主题、活动目的，根据事实特点组织材料
    • 最触动的可放开头
    • 及时分段，层次分明
    • 语言生动，注重可读性
  • 写作要领
    • 把握好报道时机
    • 处理好要素交代
    • 内容逻辑清晰、文字严谨
    • 力求新颖活泼，但不可天花乱坠

写作技巧

  • 建立选题库
    • 看剧
    • 看书
    • 与人交流
    • 上网冲浪
    • 习惯性思考看到的信息
  • 模仿写作

数字图像

图的种类

  • 位图：通过二维坐标系不同点的不同信息做出的点阵图或者说栅格图。基本的单位是像素。（.png .jpg .jpeg）
    • DPI：墨点密度，打印图片计量单位。一般200-300dpi
    • PPI：像素密度（分辨率，与画布大小无关），电子图片计量单位，结合画布大小决定最终画质。一般会默认72或者300.
  • *矢量图：呈现点雨点之间的函数关系，通过图源，将最终的曲线等比例，完全准确地呈现在屏幕上。（.ai .svg）

颜色

  • 自然光谱：自然光颜色构成了人类最大色域
  • RGB色域空间：由发光二极管组合而成的色域空间被称为RGB色域空间。
  • CMYK色域空间：不会自发光的反光材料。（需要线下进行印刷的设计

视频发布的一般流程

编辑文案：开端要好

  • 可参考同类视频，借鉴热门视频的文案
  • 巧用疑问句、反问句->激发用户好奇心，积极互动
  • 代入式文案->针对不同地域、群体、情感类型，吸引大家停留观看
  • 采用叙事方式的标题，让用户在不看视频的情况下就可以知道视频发生了什么

关联话题：站在风口，流量提升（可获得抖音官方推流）

  • 在抖音的搜索栏中搜索视频相关的话题，找到热度最高的话题并带上发布
  • 经常刷一刷抖音的热点榜，选择最近很火的模板或者音乐，这样也可以为账号带来不少流量

选择封面

  • 不可随意设置
  • 选择视频中最亮眼最吸引人的一阵作为封面
  • 好的封面可以让用户在浏览主页的时候多停留一会，翻阅更多的视频

确定发布时间：看抖音的人最多的时间

  • 时效性：每个视频和热点都有自己的时效性所以要特别留意发布时间
  • 限流：连续发布视频容易被抖音平台判定限流导致流量低迷
  • 流量时间段：抖音用户活跃时间为中午、夜晚，在此发布提高流量

提示

  • 电脑端/一定粉丝数：使用关联热点提升流量
  • 客观审视，观众比你严格

播音与Au入门

播音技巧：提打挺松

  • 提顴肌：微笑着说话，嘴角微微向上翘，同时感觉鼻翼张开，声音清亮
  • 打牙关：打开上下大牙齿（槽牙），给口腔共鸣留出空间
  • 挺软腭：半打哈欠状态

好声音

  • 控制气息（腹式呼吸3）
  • 吐字归因
    • 唇齿音（f）上齿与下唇相接，软腭上升，让气流从窄缝的空隙中泄出，摩擦成声
    • 舌根音（g、k、h、ng）发音位置靠近舌根
    • 双唇音（b、p、m）上唇与下唇接触，使语音受阻而构成的一种辅音
    • n&l
      • n——气流通过鼻腔口腔
      • l——气流只通过口腔
    • 平舌音和翘舌音
      • z c s——舌头放松抵住下牙齿
      • zh ch sh r——舌头用力卷起
    • 前鼻音和后鼻音
      • 前鼻音——舌尖用力，声音靠前
      • 后鼻音——舌根用力，声音靠后

声音如何变得更好听

  • 坚持练习口部操
  • 练习控制气息、吐音归字
  • 平时说话多加注意

序列设置

  • 菜单栏-序列-设置

  • 此处设置序列的帧速率和帧大小（分辨率）

关键帧

  • 相对于过渡帧或中间帧，是动画中最小单位的单幅影像画面，用于设置动画效果
  • 蓝色为有定义关键帧，未染色为未定义

导出视频

  • 菜单栏：文件-导出-媒体（选中时间轴面板）
  • 导出设置
    • 选择一帧做封面
    • 选择需要渲染/导出的片段（左右倒三角形）
    • 格式：视频封装格式（通常H.264，相对H265对硬件要求更低）
    • 输出名称：点击设置
    • 基本视频设置
    • 比特率（码率）设置
      • 视频每一秒所包含的数据量
      • 通常比特率越高，画质越好，视频文件体积越大
      • 1080P通常5-10Mbps
      • 编码分为CBR恒定比特率和VBR可变比特率（荐VBR）
  • 独显可加速
